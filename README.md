## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/Atharv-0700/Atharva-portfolio.git
```

Navigate to the project folder:

```bash
cd Atharva-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
